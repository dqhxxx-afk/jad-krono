# Deploy JAD KRONO 1.0

1. Extract `JAD_KRONO_1.0_Release.zip`.
2. Upload every extracted file and folder to the root of the GitHub repository connected to Vercel.
3. Commit the changes.
4. Open Vercel → project `jad-krono-v5-clean` → Deployments.
5. Wait for the latest deployment to show `Ready`.
6. Open `https://jadkrono.com` in a private/incognito tab.

## Check after deployment

- Header and Footer use the official JAD KRONO logo.
- Browser tab and iPhone preview use the JAD KRONO monogram.
- Collection brand filters work.
- Product `View` links open the correct page.
- WhatsApp, Instagram, Facebook and Google Maps links work.
- Home, Collection and Contact look correct on mobile and desktop.

## Safari favicon cache

If an older icon remains:

- close the old tab,
- open a new private tab,
- or clear Safari website data for `jadkrono.com`.
