# Deploy JAD KRONO 1.0.1

1. Back up the current GitHub repository.
2. Extract `JAD_KRONO_1.0.1_Copy_Search_Fix.zip`.
3. Copy every file and folder from the extracted directory into the local GitHub Desktop repository.
4. Choose Replace when Finder asks about existing files.
5. Commit with: `Release JAD KRONO 1.0.1`.
6. Push origin.
7. Open Vercel and wait for the new deployment to show `Ready`.
8. Test `jadkrono.com` in a fresh private tab.

## Check after deployment

- Open `/collection?brand=patek`.
- Type `126500LN` in Search.
- The active filter should change to `All Watches` and the Rolex Daytona should appear.
- Check Services: no `Private services` heading.
- Check Sell Your Watch: step 03 should read `Review`.
- Check Contact and About wording on mobile and desktop.
