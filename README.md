## Upload to GitHub

Upload the contents of this folder to your GitHub repository root:

- app/
- components/
- data/
- public/
- package.json
- next.config.js
- README.md
- .gitignore

Do not upload the parent folder itself.

## Add product images

Put images into:

public/products/

Then edit product entries inside:

data/site.js

Example:

image: "/products/daytona.jpg"
